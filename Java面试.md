## 考察数据结构

1. 链表和数组的区别： 时间和空间上谈

2. Java hashMap 是基于冲突链表实现的。 影响hash map的两个因素：初始容量（初始大小）和负载因素（自动扩容的临界值）  

3. 对于有hash冲突的键值对， 会存在冲突链表中， 新的值是插在链表头的

4. 高阶 hashset实现：由hashMap实现
